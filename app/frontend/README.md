# Frontend

**Sürüm:** 0.1.0

React tabanlı frontend uygulaması, Öğrenci Kulüp Yönetim Sistemi için.

## İndirme ve Kurulum

### Linux

```bash
# frontend dizinine geç
cd app/frontend

# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm start
```

### Windows

```cmd
:: frontend dizinine geç
cd app\frontend

:: Bağımlılıkları yükle
npm install

:: Geliştirme sunucusunu başlat
npm start
```

Uygulama [http://localhost:3000](http://localhost:3000) adresinde çalışacaktır.

## Create React App ile Başlarken

Bu proje [Create React App](https://github.com/facebook/create-react-app) ile oluşturulmuştur.

### Kullanılabilir Komutlar

Proje dizininde aşağıdaki komutları çalıştırabilirsiniz:

#### `npm start`

Uygulamayı geliştirme modunda çalıştırır.\
Tarayıcıda [http://localhost:3000](http://localhost:3000) adresini açın.

Sayfa değişiklik yaptıkça yenilenir.\
Konsolda olası lint hatalarını görebilirsiniz.

#### `npm test`

Test çalıştırıcısını interaktif izleme modunda başlatır.\
Testlerin nasıl çalıştırılacağı hakkında bilgi için [test çalıştırma](https://facebook.github.io/create-react-app/docs/running-tests) bölümüne bakın.

#### `npm run build`

Uygulamayı üretim için `build` klasörüne derler.\
React üretim modunda paketlenir ve en iyi performans için optimize edilir.

Derleme küçültülür ve dosya adları hash içerir.\
Uygulamanız dağıtıma hazır!

Dağıtım hakkında bilgi için [deployment](https://facebook.github.io/create-react-app/docs/deployment) bölümüne bakın.

#### `npm run eject`

**Not: bu tek yönlü bir işlemdir. `eject` yaptıktan sonra geri dönüş yok!**

Eğer yapı araçları ve yapılandırma seçimlerinden memnun değilseniz `eject` komutunu kullanabilirsiniz. Bu komut tek bir bağımlılığı projenizden kaldırır...
