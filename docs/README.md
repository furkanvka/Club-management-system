# Documentation

This directory contains all project documentation for the Student Club Management System, organized by development phase.

## Folders

- **feasibility-study/**: Documents related to the feasibility study, including cost-benefit analysis, technical feasibility, and project viability assessment.

- **system-analysis/**: System analysis documents including use case diagrams, data flow diagrams, and process models.

- **system-design/**: System design specifications, including architecture diagrams, database schemas, UI/UX designs, and technical specifications.
